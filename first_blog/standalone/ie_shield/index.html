As all most of us (web devs and designers) hate internet exploiter, and think that it's useless, it is useless, that's why I thought of creating IE proof.

Objective is very simple, as soon as someone with ie, trys to visit our site, we put up an ie_shield like this : I know this can dismiss some possible future clients, but at this point, recall the main motto of your brand. I'm sure its "CLIENT SERVICE". And I'm sure that you dont want to leave your client with crap like ie, so its highly recommendable.

And setting it up is fairly simple. Just 4 steps very simple steps.

  1. Creating i_hate_ie.css
     All you need to do is save the following code as i_hate_ie.css or whatever you feel like. Make sure to check that you have id="wrapper" and not class="wrapper".

    /*IE CSS TO PREVENT IE TO VIEW MY WEBSITE*/
    #wrapper{display:none;}
    .extra{
    color:#666;
    font-size:95%;
    font-style:italic;
    list-style:none;
    margin:5px 0 0 20px;
    }
    #ie_proof{
    background-color:#E0EBEB;
    margin:20px auto;
    padding:200px 100px 170px 200px;
    position:absolute;
    width:450px;
    }
    #ie_proof table{margin:30px 0 30px -15px;}
    #ie_proof a:hover{background-color:#480006;}

    Save this as i_hate_ie.css in your website directory and call it into everypage using [if IE] statement.
  2. The html
     Now you need to make each page capable of fighting IE. So we contact our old friend html. Add this code at the top of every page of your site, or atleast the home page.

    <!--IE-PROOF-->
    <div id="ie_proof">
    <p>I won't allow IE to see my website</p>
    <p>Download a browser, IE is not one.</p>
    <table><tr>
    <td><a href="http://www.google.com/chrome" target="_blank"><img src="chrome.png" alt="Google Chrome"/></a></td>
    <td><a href="http://www.mozilla.com/en-US/firefox/firefox.html" target="_blank"><img src="firefox.png" alt="Mozilla Firefox"/></a></td>
    <td><a href="http://www.apple.com/safari/" target="_blank"><img src="safari.png" alt="Apple Safari"/></a>/td>
    </tr></table>
    <p class="extra">You may call me arrogant, but that according to me is the best treatment <del>crap/del> <ins>porky crap</ins> like IE can get.</p>
    </div>
    <!--end:IE-PROOF-->

  3. The graphics
     Download the graphics folder here and save the graphics into your directory. ../downloads/graphics.zip. (You can also get them from my directory.)
  4. Hiding IE-PROOF in your normal css.
     Hide IE-PROOF so that non IE users are not stopped by the shield.Just add the following code to your master.css

    #ie_proof{display:none}

    5. Now you can add an optional IE-PROOF badge.

    Just add the following css to your master.css

    #badge{
    background-color:#666;
    color:#fff;
    float:right;
    font-size:80%;
    margin:10px 0 0 0;
    position:relative;
    padding:4px;
    -webkit-border-radius:4px;
    -moz-border-radius:4px;
    border-radius:4px;
    }
    #badge:hover{background-color:#555;}
    #badge span{
    color:#33ccff;
    font-size:105%;
    font-weight:bold;
    }


    And here's the html. Add it at the top inside the #wrapper div.

    <!--BADGE-->
    <div id="badge">
    <span>IE{/span>-PROOF
    </div>
    <!--end:BADGE-->
